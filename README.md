# Booking System

A booking management system for custom services

### Apps
Application servers
#### Main Server
- Handles user authentication and user services

#### Booking Server
- Process and monitors all booking request and services

#### Payment Server
- Handles all payment transactions and services

### Libs
Application libraries 
#### libcommon
- Common library that holds shared resources between applications
